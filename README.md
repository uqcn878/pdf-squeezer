## 它到底能干什么

核心功能其实挺简单——把 PDF 压小。但它做得比你想象的精细。

拖进去，选一个压缩档位，就完了。三档预设（低、中、高压缩）覆盖了大部分场景，懒人友好程度拉满。要是你对图片质量有要求，还可以自己调分辨率和压缩参数，创建专属配置保存起来反复用。

另外有几个实用功能值得单独说一下：

**并排对比预览**：压缩前后效果同屏展示，不满意直接换参数，不用反复导出对比。

**批量处理**：把一整个文件夹的 PDF 一次性全丢进去，它挨个处理，不用盯着一个个等。

**提取图片 / 文本**：付费版支持从 PDF 里单独提取图片或文本内容，有时候挺用得上的。

**删除元数据**：PDF 里经常藏着作者信息、修改记录、地理位置等看不见但确实存在的信息。压缩时一并清除，免得不小心泄露。

**自动化集成**：支持 macOS 自动操作（Automator），可以配置成收到 PDF 自动压缩保存，适合有批量处理需求的工作流。
